#TodoApi
=========================

<p>
    A Restfull API made with Node.JS, Express by using MONGODB as database and including Authentication layer with JSONWebToken
</p>



### Installation

- git clone https://gitlab.satoripop.com/joel.zulbal/silva.git
- npm install

In Development
- npm run watch 

In Production
- npm run production
